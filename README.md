# Personal Finance Calculator

This project is a simple personal finance calculator built in Python to track income, expenses, and financial balance.

## Features

- Add income and expenses
- Track financial balance
- Store financial data in JSON format
- Basic financial data management

## Technologies Used

- Python
- JSON data storage

## Project Structure

finance_manager.py – main program logic  
finance_core.py – core financial calculation functions  
finance_data.json – example financial data storage  
test_finance_core.py – basic unit tests

## Purpose

This project was developed as part of my computer science coursework to practice programming concepts and financial data management.
